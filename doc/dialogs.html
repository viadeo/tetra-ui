Popins

Popins, or modal dialogs, or pop-up windows now are a common user experience design pattern. Here are some samples and the code to launch a popin.

Launch popin sample 1

Launch popin sample 2

Display loaded popin 

<a data-comp="popin"
  data-event="click"
  data-popin="show"
  data-target-url="[url]"
  class="bootnode btn btn-primary">
  Launch popin
</a>

Here are all the required HTML attributes to launch a popin.

Attribute Description
data-comp="popin" Calls the required JavaScript component
data-event="click" Displays popin on click
data-popin="show"
data-popin="close" 		Shows or closes the popin
data-target-url="[url]" Fetches the HTML fragment located at the target URL
data-target-id="[id]" Displays a popin with content of the element #id
class="bootnode ..." Instantiates the JavaScript framework

A code sample of the popin itself: the target HTML fragment loaded by the component. Whether it's an Ajax call or the content of the specified id, it's the same markup.

<div class="gr"><!-- Initializes the grid -->

<!-- Base popin class, width set to two thirds of page width -->
<div class="popin gu-2of3">
  <div class="popin-header">

    <!-- Close button -->
    <button data-comp="popin" data-event="click" data-popin="close" class="bootnode close">
      Close
    </button>

    <h5>Popin header</h5>
  </div>

  <div class="popin-body">
    <p>Popin sample body</p>
  </div>

  <div class="popin-footer">
    <a href="" class="fr mts">Footer link</a>
    <a href="#" class="btn btn-primary btn-large">Footer button</a>
  </div>
</div>

</div>

Events listened and triggered by the popin controller.

Events listened by the controller Description
page.notify('popin: set content', { url: '[URL]' }); Displays or updates a popin with the HTML code at the given URL with an Ajax call.
page.notify('popin: set content', { html: '[code]' }); Displays or updates a popin with the given HTML code.
page.notify('popin: set content', { id: '[id]' }); Displays or updates a popin with HTML code in the DOM. This code is the HTML content of the element with the given id.
Events triggered by the controller Description
page.notify("popin: success", obj); The popin controller sends this event when the popin is successfully displayed or updated. The variable obj contains the HTML code which is contained in the popin.

Highlights

Highlights are a way to put focus a specific item on a page. You can use it to feature a preview or a tutorial step on the page.

Interesting box

This box looks really interesting, you shoud highlight it.

Highlight me ! 
<a data-comp="highlight"
    data-event="click"
    data-highlight="[css selector of the target]"
    class="bootnode btn btn-primary">
    Highlight me !
  </a>

Events listened by the highlight controller.

Events listened by the controller Description
page.notify('highlight: display', [css selector of the target]); Highlight the element provided as an argument on the page.
page.notify('highlight: close'); Remove the current highlight off the page.
